vue学习笔记4-3

vue学习系列3

v-if v-else-if v-else

一般写在computed中

补漏html知识

label for=’user’用户账号 for与id对应
input type=’text’ id=’user’ 点击label内容是会提示你在id中输入

v-show
与v-if语法形式基本相同，但是当v-if为false时元素不会存在于dom中，v-show为false时只是给元素增加一个行内样式：display: none
如果显示与隐藏切换频率高时使用v-show
v-for
如果需要获取索引需要改写为 (item,index) in array
如果遍历对象属性改写为 (value,key) in obj，如果只写一个默认是value

数组与响应式

数组响应式方法
push pop shift（删除数组中第一个元素）unshift(‘value’)（数组开头添加元素）
splice（删除、插入、替换元素）（删除位置,删除数量）（插入位置,0,插入值）（替换位置,1,替换值）sort reverse
非响应式
array[index]=’change’（可以使用Vue.set(array,index,value)或者splice）

小案例图书购物车

javascript高阶函数的使用

let array=[2,3,5,67];
let newarray=array.filter(function(n){ return n<10});//得到小于10的元素
let newarray=array.map(function(n){ return n*2});//所有元素乘2

表单绑定v-model

双向绑定，改变text中的内容，message也会改变
v-model相当于:value加上@input指令

结合checkbox，单选框对应bool值，多选框对应数组类型

修饰符
lazy v-model.lazy 取消实时绑定，失去焦点或回车时更新
number v-model默认是string v-model.number绑定为number
trim v-model.trim去除前置和后置空格

组件化

将复杂问题分解为小问题。将一个页面拆分为一个个小的功能块，每个块完成自己独立的功能，有利于页面的维护和管理，扩展性更强

组件的使用步骤

1、创建组件构造器
2、注册组件
3、使用组件
es6语法补充：`` 可以换行
Vue.extend()创建一个组件构造器。传入template代表自定义的模板，该模板就是用到组件的地方要显示的HTML代码
Vue.component()将组件构造器注册为一个钻进传递两个参数，一个是组件标签名一个是组件构造器
组件必须挂载在某个vue实例下否则不会生效

全局组件和局部组件

全局组件可以在多个vue实例下使用，局部组件挂载在某个vue中在指定vue实例中使用

父组件和子组件

cpnc2是父组件，cpnc1是子组件，cpnc1不能再vue实例中单独使用如果要使用必须也注册在vue实例中

注册全局组件语法糖 Vue.component(‘cpnc1’,{template: })
注册局部组件语法糖 components: {‘cpnc1’: {template: } }

组件模板抽离写法

第一种方式，也可以使用template标签，形式跟之前一样

组件数据存放问题

组件不能直接访问vue实例中的数据，组件自身也有data属性用于存放数据，此data属性不是对象
类型而是一个函数，返回一个实例对象保存数据
data是一个函数，因此组件可以实现复用多个组件数据相互之间不会影响因为返回的是不同的数据对象

父子组件的通信

子组件不能引用父组件或者vue实例的数据

通过props(properties)向子组件传递数据

1、字符串数组，数组中的字符串就是传递是的名称
2、对象，对象可以设置传递时的类型，也可以设置默认值
如果以对象传递可以指定类型 String Number Boolean Array Object Date Function Symbol
如果类型是数组或者对象默认值必须是一个函数 default(){return []|{}}

props的驼峰标识
cpn标签中不支持驼峰写法 例如cName无效需要改为c-name chuldMessage改为child-message

子组件通过自定义事件向父组件发送消息

子组件需要向父组件传递数据是就要用到自定义事件
v-on不仅可以用于监听DOM事件也可以监听组件的自定义事件
子组件中通过$emit()来触发事件 父组件中通过v-on来监听子组件事件
流程：首先监听子组件的btn方法，当点击按钮时触发，在btn方法中向父组件传递事件itemclick，父组件监听此事件
