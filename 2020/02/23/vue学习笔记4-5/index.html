vue学习笔记4-5

vue学习系列5

webpack图片文件

terminal中输入指令 npm install –save-dev url-loader
当图片大于limit是需要输入 npm install file-loader –save-dev

在webpack.config.js中配置

大于limit的图片会打包到dist中这时需要该路径在output中加入 publicPath路径

webpack配置vue

输入指令 npm install vue –save
在webpack.config.js文件中配置

注意

id写在前面，script引用写在后面

Vue的template

在实际开发中，可能只有一个html文件，Vue绑定的div中不写内容，把内容写在Vue的template中
template会替换el

进一步分离，创建一个App.js文件，默认导出


将App中的模板和js代码分离，内容写在vue文件中
安装vue-loader npm install –save-dev vue-loader vue-template-compiler


App.vue中可以引用Cpn等组件，注意所有内容都必须写在标签中，要不然可能报错



在webpack.config.js文件中配置vue

注意

如果修改package.json文件，那么之后 必须执行npm install指令

版权plugin

BannerPlugin webpack自带插件声明版权

打包html的plugin

发布项目时需要将html文件打包到dist中
安装HtmlWebpackPlugin插件 可以自动生成index.html文件，将打包的js文件通过script标签插入body中
npm install html-webpack-plugin –save-dev
修改配置文件

js压缩的plugin

对js代码进行压缩，删去空格修改变量名
npm install uglifyjs-webpack-plugin@1.1.1 –save-dev
修改配置文件

注意

IDEA中out of memory，暂时不使用本地服务器以下只了解即可

搭建本地服务器

npm install –save-dev webpack-dev-server@2.9.1
修改配置文件

在package.json中scripts标签中加入 “dev”: “webpack-dev-server “
启动服务器 npm run dev

webpack配置文件的分离

开发与生产时配置文件不同，创建多个配置文件，base.js放公共配置，prod.js放生产配置，dev.js放开发配置
安装 npm install webpack-merge,可以合并配置文件
引入 const webpackMerge=require(‘webpack-mrege’)
例如合并base.js与prod.js 在prod.js中引入base.js,然后module.exports=webpack(base,{})



修改package.json文件，删除webpack.config.js文件
