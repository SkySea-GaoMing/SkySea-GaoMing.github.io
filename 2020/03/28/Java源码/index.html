不需要手动导入

在Java中有些类需要手动导入(比如各种集合类，Date类)有些类不需要手动导入(比如Integer,
String,Object),为什么会出现这种情况？有些类经常使用，所以编译器自动帮助我们导入这些类。

  • import java.util.Date 单类型导入，直接导入我们需要的类
  • import java.util.* 按需类型导入，并不是导入所有该包下的类，当我们使用某个类时会按需
    导入
