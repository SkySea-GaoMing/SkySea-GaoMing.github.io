Shiro

先学习一下Shiro，一方面是因为Spring Security比较复杂，另一方面我目前
学习的White-Jotter项目后端用的安全框架就是Shiro

Shiro简介

Shiro可以帮助我们完成：认证、授权、加密、会话管理、与 Web 集成、缓存等

  • Authentication：身份认证 / 登录，验证用户是不是拥有相应的身份
  • Authorization：授权，即权限验证，验证某个已认证的用户是否拥有某
    个权限即判断用户是否能做事情，常见的如：验证某个用户是否拥有某个角
    色。或者细粒度的验证某个用户对某个资源是否具有某个权限
  • Session Management：会话管理，即用户登录后就是一次会话，在没有退
    出之前，它的所有信息都在会话中
  • Cryptography：加密，保护数据的安全性，如密码加密存储到数据库，而
    不是明文存储
  • Caching：缓存，比如用户登录后，其用户信息、拥有的角色 / 权限不必
    每次去查，这样可以提高效率
  • Remember Me：记住我，这个是非常常见的功能，即一次登录后，下次再来
    的话不用登录了
